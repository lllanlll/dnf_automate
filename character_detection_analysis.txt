"""
DNF角色识别方案分析

方案1: 角色模板匹配
- 优点: 最精确

- 适用: 大多数情况

方案3: 角色名字识别
- 优点: 非常稳定
- 缺点: 需要OCR，性能开销大
- 适用: 高精度需求

方案4: 屏幕中心假设
- 优点: 简单，无需识别
- 缺点: 不准确，角色可能偏移
- 适用: 简单场景

方案5: 移动检测
- 优点: 通过移动轨迹推断角色位置
- 缺点: 复杂，需要连续帧分析
- 适用: 动态场景

推荐方案: 方案2 + 方案4 结合
